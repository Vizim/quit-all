A minimalist macOS utility to instantly quit all running applications.

![Version](https://img.shields.io/github/v/release/Vizim/quit-all)
![Platform](https://img.shields.io/badge/platform-macOS-lightgrey)

## Features
- **One-Click Cleanup:** Instantly closes all user applications.
- **Custom Icon:** Features a minimalist iOS 8 styled Control Center icon.
- **Lightweight:** Built using native macOS Automator workflows for zero overhead.

## Installation
1. Download the latest \`QuitAll.zip\` from the [Releases](https://github.com/Vizim/quit-all/releases) page.
2. Unzip and move \`Quit All.app\` to your \`/Applications\` folder.
3. (Optional) Drag it to your Dock for quick access.
